演習1-4: HTMLのさまざまな要素

参考資料「HTMLテキストの要素」，「ハイパーリンク」，「画像の埋め込み」，「その他の要素の紹介」を参考にして次の演習に取り組んでください．

  1. practice-html/elements.html を Visual Studio Code で開きます
  2. 「見出し」で始まる行は1番大きな見出しの要素にしてください．
  3. 「小見出し」で始まる行は2番目に大きな見出しの要素にしてください．
  4. 拓殖大学の5学部を番号なしのリスト要素にしてください．
  5. カレーの作り方を番号付きのリスト要素にしてください．
  6. 「演習1-3のHTMLファイル」と「index.html」をハイパーリンクにしてください．
    • ともに href 属性には相対URLを指定すること
  7. 宇宙望遠鏡は説明リスト要素にしてください．
  8. 「(JSWTの画像)」を img 要素にしてください．
    • 画像ファイルは practice-html フォルダに存在する JWST...png です．（ファイル名は自分で確認すること）

（完成イメージ）

（注意事項）

  • 完成した HTML ファイルを Firefox で開くこと
  • ハイパーリンクをクリックすると演習1-3で作成したページ，index.html が表示されることを確認すること
